# 应用图标制作说明

PWA需要两个尺寸的图标：
- icon-192.png (192x192像素)
- icon-512.png (512x512像素)

## 快速创建图标的方法

### 方法1：使用在线工具
1. 访问 https://favicon.io/ 或 https://realfavicongenerator.net/
2. 上传你的logo或选择一个emoji（如✈️）
3. 下载生成的图标文件
4. 将文件重命名为 `icon-192.png` 和 `icon-512.png`
5. 放到本目录下

### 方法2：使用设计软件
1. 使用 Photoshop、Figma 或 Canva
2. 创建 512x512 的画布
3. 设计简洁的图标（建议使用飞机✈️或行李箱🧳元素）
4. 导出为 PNG 格式
5. 缩放一份为 192x192

### 方法3：临时使用占位图标
如果只是测试，可以暂时使用纯色方块：
- 使用 https://placeholder.com/ 生成占位图
- 或用画图工具创建紫色渐变方块

## 图标设计建议
- 使用简洁的图形
- 背景色与应用主题色一致（#667eea 紫色）
- 避免过多细节（小图标上看不清）
- 推荐使用 ✈️ 飞机或 🧳 行李箱作为主元素

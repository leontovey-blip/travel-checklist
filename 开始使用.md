## 📱 三步上线（最简单）

### 方法1：Netlify拖拽部署（最快，2分钟）

```
1. 访问 https://app.netlify.com/drop
2. 将"旅行备忘录软件"文件夹拖进去
3. 复制生成的URL，在手机浏览器打开
4. 点击菜单 → "添加到主屏幕"
5. 完成！像App一样使用
```

### 方法2：GitHub Pages（推荐）

运行以下命令：
```bash
cd "/Users/leontovey/Desktop/旅行备忘录软件"
./deploy-github-pages.sh
```

然后按照提示操作即可。

### 方法3：Vercel

1. 访问 https://vercel.com
2. 用GitHub登录
3. 导入项目，一键部署

## 💡 小贴士

1. **首次使用**：需要先在线打开一次，之后就能离线使用了
2. **数据备份**：数据保存在浏览器本地，清除浏览器数据会丢失
3. **分享应用**：部署后将链接分享给朋友，他们也能添加到手机使用
